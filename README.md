# Magic Suspense

Magic Suspense is a Flutter application that showcases animated suspense effects and interactive UI elements. This project demonstrates how to work with animations, transitions, and responsive widgets in Flutter.

## Features

- Smooth animations and transitions  
- Interactive suspense‑oriented UI  
- Built with Flutter for cross‑platform support (Android, iOS, Web)  
- Easy‑to‑extend code structure  

## Tech Stack

- Flutter  
- Dart
